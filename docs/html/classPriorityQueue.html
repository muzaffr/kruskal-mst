Detailed Description

template<class T>
class PriorityQueue< T >

This class implements a priority queue using a heap as the data structure.
It supports empty initialization as well as initialization from a C-style or a C++-style range.
It comes with friend functions for heapsort, utilizing the heap coroutines.
